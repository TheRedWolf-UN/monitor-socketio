# Actividad
En un principio, se trata de una página web que python va a ejecutar.
Tienes que ejecutar *app.py* y *api.py* a la vez.
En la página, debe aparecer una página de monitoreo que detecta cada vez que el archivo *tarea1.txt* sea manipulado.
Ya debería estar funcionando, recomiendo que ajustes la configuración de la IP.
Para acompletar la actividad, debes implementar un sonido que avise cada vez que el archivo sea modificado o consultado; puedes usar un audio de YouTube (el profe le quería poner los gemidos del Homero Simpson en mi página).
La implementación del sonido lo debes hacer atraves del directorio *static*.
